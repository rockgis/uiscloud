WFS 서버 요청

이 예제는 WFS 서버에 요청하는 방법을 보여줍니다.

왼쪽 지도는 Ajax 요청을 사용하여 WFS형식으로 데이터를 검색하며, 오른쪽 지도는 JSONP 기술을 사용하여 데이터를 검색합니다.

두 지도는 ol.loadingstrategy.createTile 클래스를 사용하여 같은 타일 벡터 데이터를 요청합니다.
